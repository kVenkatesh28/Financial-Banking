# FinGuard - Credit Card Churn Prediction

## Project Overview
FinGuard is a machine learning project developed in the Finance & Banking domain to predict customer churn for credit card users. The goal is to identify customers who are likely to leave the bank and help improve customer retention strategies.

## Problem Statement
Banks often face challenges in retaining customers. This project aims to analyze customer data and build a predictive model that can identify potential churners before they leave.
